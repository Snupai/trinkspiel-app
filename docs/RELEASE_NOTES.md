# Release Notes

## 3.0

Seemops Trinkspiel 3.0 is the first release-ready build pass.

Highlights:

- Offline game flow with random card draw, skip, undo, and deck reset
- 128 built-in cards across Classic Starter, Chaos Pack, Couples Pack, Preparty Pack, WG Abend Pack, Bar Runde Pack, Spieleabend Pack, and Feierabend Pack
- Built-in content review guardrails for drink limits, pacing prompts, and high-risk prompt wording
- Custom card manager with add, edit, delete, search, sort, pack/category filters, pause/activate, and bulk visible-card actions
- Import/export/share for card packs, including a confirmation preview before import
- Full JSON backup/restore for cards and settings, including a confirmation preview before restore
- Offline device-transfer package sharing/import as a JSON attachment, including direct Android share/open handling
- Pack templates that create paused, editable draft cards with category and intensity guidance
- Game modes, intensity levels, pack toggles, and custom category selection
- First-run setup defaults to low intensity, with higher intensity left as an explicit choice
- First-run setup can choose any built-in pack as the starter pack or start empty
- Players, teams, player rename, scoreboard, and manual score/drink adjustment
- Local session recap with current round totals, recent drawn cards, and top player standings
- Final UI cleanup for clearer card metadata, wrapped player chips, safer long settings labels, and more stable header stats
- Theme, dynamic color, custom drink wording, safety notice reset, and privacy/safety settings
- In-app app/version/legal section with view/share privacy policy actions
- First-run 18+ setup before gameplay
- Local diagnostics report export/share, focused last-issue export/share, and support request preparation from settings
- Polished theme colors, tighter shapes, and faster access to the main card draw flow
- Release docs, privacy policy page, static privacy-hosting bundle helper, data-safety draft, ASAP backlog, signing helper scripts, non-secret signing handoff helper, manual-QA evidence packet, release-owner brief, store graphics, launcher icons, upload package script, and Play Store checklist

Known release blockers:

- A real release signing configuration is still required before Play upload.
- Final real-device QA, hosted privacy-policy URL, and signing should be completed before public rollout.
